## Keyword Research Template

For each article, identify:
- **Primary Keyword**: Main focus (e.g., "professional posing tips")
- **Secondary Keywords**: 3-5 related terms (e.g., "modeling poses," "photography posing")
- **Long-tail Keywords**: 2-3 specific phrases (e.g., "how to pose for fashion photography")
- **Search Volume**: Estimated monthly searches
- **Difficulty**: Keyword difficulty score (1-100)
- **Search Intent**: Informational, navigational, transactional

## SEO Checklist for Each Post

### Before Publishing
- [ ] Primary keyword in title (preferably at beginning)
- [ ] Primary keyword in first 100 words
- [ ] Keyword density 1-2% (natural usage)
- [ ] H2 and H3 headings include semantic variations
- [ ] Meta title optimized (55-60 characters)
- [ ] Meta description compelling (150-160 characters)
- [ ] Featured image optimized (compressed, named, alt text)
- [ ] All images have descriptive alt text
- [ ] 3-5 internal links to relevant pages
- [ ] 2-3 external links to authoritative sources
- [ ] Clear CTA at the end
- [ ] Mobile-friendly formatting
- [ ] Readability score 60+ (Hemingway Editor)
- [ ] Grammar checked (Grammarly)
- [ ] Schema markup added (Article schema)

### After Publishing
- [ ] Submit URL to Google Search Console
- [ ] Share on social media platforms
- [ ] Add to email newsletter
- [ ] Monitor comments and engage
- [ ] Track rankings in rank tracker
- [ ] Review analytics after 7 days
- [ ] Update/refresh after 90 days if needed
